# 🚀 Capabl Agentic AI Internship – JobPilot AI

## 👥 Team: JobPilot AI

JobPilot AI is an AI-powered job search and career assistance platform developed as part of the Capabl Agentic AI Internship.

The platform aggregates job listings from multiple platforms, allows filtered job search, displays structured results, and redirects users to official job application pages.

Future enhancements include:
- LLM-based interview question generation
- Resume-based job matching
- ATS-friendly resume analysis
- Job bookmarking system
