## 📖 Project Overview

JobPilot AI is an AI-powered job search and career assistance platform designed to help users find relevant job opportunities efficiently.

The system allows users to search for jobs based on multiple filters and fetches real-time job listings from various professional platforms.

---

## 🎯 Week 1 Implementation

In Week 1, we successfully developed and demonstrated the core job search functionality with live data fetching and structured result display.

---

## 🔍 Key Features Implemented

### 1️⃣ Smart Job Search
Users can search jobs based on:
- Job Title
- Location
- Experience Level
- Platform (LinkedIn, Indeed, AngelList, etc.)

---

### 2️⃣ Multi-Platform Job Aggregation
The system fetches job listings from multiple platforms such as:
- LinkedIn  
- Indeed  
- AngelList  
- Other supported job portals  

---

### 3️⃣ Structured Result Display

When the **Search** button is clicked:

- Data is fetched from real job platforms
- Results are displayed in a structured table format

The table includes:

- Job Title
- Company Name
- Location
- Platform Name
- Experience Required
- Posted Days (e.g., 2 days ago)

---

### 4️⃣ Expandable Job Details

Below the job table:

- A dropdown menu allows users to view full details of a selected job listing
- Detailed job description is displayed for better clarity

---

### 5️⃣ Direct Apply Feature

Each job listing includes an **Apply Button**  
When clicked:

- User is redirected to the official job platform website
- Users can apply directly on the original platform
